Properties

The Properties window provides the opportunity to edit and view information about the project.



  • Project Name: Displays the Name of the project. To change the name of the project go to File and select Save As. Use Save As to rename your project.

  • Project Description: Enter a description for this project by typing in the Project Description text box. After saving the project, the description will appear in this location, and can be edited at any time.

  • Project File: Displays the location that the project is stored. If the entire path name is not visable drag the window wider to reveal the entire name.

  • Created By: Displays the user name of the person who created the project.

  • Created Date: Displays the date and time when the project was first created.

  • Project Notes: Enter notes for this project by clicking in the text box. After saving the project, the notes will appear in this location, and can be edited at any time.

  • Monetary Unit: Displays the monetary units used in the project. The default units are in thousands of dollars, but can be changed by using the dropdown box to dollars, or millions of dollars.

  • Unit System: Select the unit system for the project. The default is the English system but can be changed using the dropdown box to SI units.

  • Surveyed Year: Displays the year the data was surveyed. This value can be changed by clicking in the text box.

  • Updated Year: Displays the updated year. This value can be changed by clicking in the text box.

  • Updated Price Index: Displays the updated price index.
