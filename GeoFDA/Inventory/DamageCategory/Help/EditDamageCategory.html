Edit Damage Category

You can edit the damage categories using this form.



  • Click Add to add a new row.
  • Click Remove to remove the selected row(s).
  • Double click a cell to make changes to the data inside.
  • When finished click the Ok button.
  • Press the Cancel button to cancel. This will NOT save any changes.

Example 1: Selecting Multiple Rows


Select a row by clicking to the left of the first cell in the desired row. Multiple rows can be selected by clicking and dragging to the left of the cells.



Example 2: Adding Rows


To add a row click the Add button. If a row has not been selected a new row will appear at the top. If a row has been selected the new row will be inserted above the selected row.



The red exclamation point and outline indicate an error with that field. In this case the error is because the Name field cannot be empty. Double click the red cell and add the desired data.



Example 3: Removing Rows


Select the row or rows to be removed and press Remove.



The row will be removed.
