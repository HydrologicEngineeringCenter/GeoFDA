Import From NSI

Importing structures from the National Structure Inventory (NSI) is an easy way to get structures quickly. Supply a shapefile that will be used to get all the structures within that extent.



  1. Enter the name for the structure inventory.

  2. Select or browse to the shapefile that contains the structure inventory shapefile.

  3. Radio Buttons:

    1. Select From Web for Geo-FDA to access the NSI website to import the structures that are contained by your Study Are6a Shape.

    2. Select From User Defined Shapefile to import a previously downloaded NSI shapefile.

Example




After clicking the Ok button, the name of the structures will appear in the Study Tree.
