Terrain Importer

The Import Terrain dialog is used to import a terrain file into the project. GeoFDA accepts three file formats: GeoTiff(.tif), ESRI floating point binary (.flt), and virtual format (.vrt).



  1. Terrain Name: Enter the name of the terrain.

  2. Terrain Path: Enter or browse to the directory where the terrain is located.

Example




After clicking the Ok button, the name of the terrain will appear in the Study Tree.
