Index Location Editor

The Index Location Editor provides a way to edit the damage reach index points.



  1. Select the Reach Name from the Define Index Locations window.

  2. Enter the River Station number.

  3. Enter the Invert value.

  4. The form will provide eight rows. Enter the corresponding data for that particular reach and station.

  5. In order to create as FDA Import File all of the reaches need index location data defined.
